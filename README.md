**Student Management System**

**Project Description**

Developed a Student Management System using Python and the Django framework to efficiently manage student records through a web-based application. The system allows administrators to perform CRUD (Create, Read, Update, Delete) operations on student information, departments, and courses using a secure admin interface.

**Key Features**
Secure admin login and authentication.
Add, update, delete, and view student records.
Manage departments and courses.
Search and filter student information.
User-friendly web interface.
Database integration using Django ORM.
Responsive templates using HTML, CSS, and Bootstrap .

**Technologies Used**
Frontend: HTML, CSS, Bootstrap

Backend: Python, Django

Database: SQLite 

ORM: Django ORM

Tools: VS Code, Git, GitHub
